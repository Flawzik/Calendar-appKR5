- Базовый Express-сервер
- Маршруты: GET, POST, DELETE
- Работа с `req.params`, `req.query`
- Обработка JSON тела запроса
- Собственный middleware (логирование)
- Раздача статики через `express.static()`
- Логика разнесена по модулям (routes + controllers)
- Структурированный репозиторий
